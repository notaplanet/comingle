Spreadsheet Access

Our team mailing list is notaplanet-active@googlegroups.com. The email that is subscribed to this list is the one that will give you access to our Google docs.

!! If you can't access the sheets:
  1. Check that you're logged in with the email address subscribed to the list.
  2. If you need to change the email that's subscribed (e.g. gmail), go here to add your new email address.
  3. If you're not subscribed to our list and need to be added before hunt, email notaplanet-command@googlegroups.com with the subject "SIGN-UP REQUEST"
  4. If you're not subscribed to our list and need access after hunt has started, please contact the person who invited you to have them log you in using their credentials.

Shift Captain Duties

The shift captain is the first point of contact for minor questions or problems: ​ The main responsibility is to make sure that people are signed up for events or other time-sensitive actions. It is recommended to also use slack's "reminder" feature, e.g.: ​ /remind #general "The Towers of Hanoi event is starting in 30 minutes" at 2:30pm ​ If any problem comes up that you can't handle, notify the command team. Any new puzzles should be scraped from the hunt website by Ceres within 3-5 minutes. If this is not working, manually inform Ceres about any new puzzles (and let the tech team know!). First, type "@Ceres help" in slack to get a reminder of the syntax. (If you don't get a reply, then Ceres is off!) Then use the newpuzzle command to create a new puzzle, e.g.: ​ @Ceres newpuzzle "Wizardry" "The Towers of Hanoi" https://mitmh23.com/wizardry/hanoi ​ This command will automatically create Google sheets, comingle room, slack channel, etc. within about 3 minutes. Sometimes there are special events or other sorts of puzzles that don't have a dedicated page on the official MH website: you can use the newpuzzle command for such circumstances. Note that the url for each puzzle must be unique. ​
